# Task 03 - Sudoku Solver

This folder contains a program to automatically solve Sudoku puzzles.

## How to Run

```bash
python sudoku_solver.py
```
## Description

The program takes a 9x9 Sudoku grid with 0 representing empty cells.

Uses a backtracking algorithm to fill in missing numbers.

Prints the solved Sudoku grid.

## Example
Input grid:
```
[5,3,0,0,7,0,0,0,0]
[6,0,0,1,9,5,0,0,0]
...
```
Solved grid:
```
[5,3,4,6,7,8,9,1,2]
[6,7,2,1,9,5,3,4,8]
...
```
